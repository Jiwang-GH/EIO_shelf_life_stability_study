### EIO Shelf Life Stability Study

This repo contains code pertaining to statistical analysis performed under the EIO shelf life stability study protocol below.

Protocol is Document ID PLP-001860: Guardant360 CDx Enrichment Index Oligo (EIO) Shelf-Life Protocol for Plated Components.

Report is Document ID RPT-007807 R1: Guardant360 CDx Enrichment Index Oligo (EIO) In-house Plate (GHM0355) Shelf Life Study Report. Current link is https://docs.google.com/document/d/1AIAzfhPQL0fTUlw9lR7MNBs4WEyl-ESE/edit
